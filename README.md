# Gerenciador de Tarefas CLI em Python

Projeto desenvolvido em Python para gerenciamento de tarefas via terminal, com foco no aprendizado de lógica de programação, organização de código e persistência de dados.  
Este é meu primeiro projeto completo, construído durante meus estudos iniciais em programação.

## Funcionalidades
- Criar tarefas
- Listar tarefas
- Atualizar status das tarefas
- Remover tarefas
- Persistência de dados para manter as informações salvas entre execuções

## Tecnologias Utilizadas
- Python 3
- SQLite
- Git e GitHub
